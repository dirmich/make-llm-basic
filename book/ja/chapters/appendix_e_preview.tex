\chapter{高度なトピックのプレビュー}
\label{app:advanced_preview}

この付録では、2巻\textit{Make LLM-advanced}で取り上げる高度なトピックを簡単に紹介します。 1冊を終えた読者が次の学習方向をつかむのに役立つ。

\section{分散学習}

1巻モデルは単一GP​​U/CPUで学習した。しかし、70B以上のモデルは単一のGPUメモリに入らない。

\keyterm{DDP (Distributed Data Parallel)}：モデルのレプリカを各GPUに置き、データのみを分割します。最も単純です。

\keyterm{FSDP (Fully Sharded Data Parallel)}：パラメータ、グラデーション、オプティマイザの状態をすべてシャーディング。 70B+モデル学習可能。

\keyterm{ZeRO}：DeepSpeedの段階的なシャーディング。 ZeRO-1/2/3で徐々にメモリを節約。

\keyterm{パイプライン並列}：モデルを階層別にGPUに分割。ノード間通信に適しています。

\keyterm{テンソル並列}：単一レイヤーの重みを分割します。 Megatron-LM方式

\section{最新のアーキテクチャ}

1巻は2017年のオリジナルトランスフォーマーに近い。最新のLLMはより効率的なコンポーネントを使用しています。

\keyterm{RoPE (Rotary Position Embedding)}：複素数回転で位置エンコード。 extrapolationは優れており、長いコンテキスト処理にガラス。 LLaMA、Mistralを使用。

\keyterm{GQA (Grouped Query Attention)}：KVヘッドを減らして推論メモリを節約します。 LLaMA-2 70BがGQA-8を使用。

\keyterm{SwiGLU}: GLU バリアントの活性化による FFN パフォーマンスの向上.

\keyterm{RMSNorm}: LayerNorm の軽量バージョン。計算量 10$\sim$20\% 減少。

\keyterm{MoE (Mixture-of-Experts)}：複数の専門家のうち、トークンごとに上位K個のみを有効にします。 Mixtral 8x7Bが47Bの全パラメータで13Bアクティブに動作。

\section{量子化と推論の最適化}

\keyterm{INT8/INT4 量子化}: 重みを低ビット整数に変換することでメモリを4$\sim$8倍節約。推論コスト決定的削減

\keyterm{AWQ}: アクティブ値統計による重要なチャネルの保存。 INT4でも高精度。

\keyterm{KV Cache}：自己回帰生成で以前のK、Vを再利用。各ステップの複雑さ$O(L^2) \to O(L)$。

\keyterm{PagedAttention}: vLLM のページ単位の KV 管理。マルチシーケンススループット2$\sim$4倍向上。

\section{ソートとファインチューニング}

1巻モデルは「次のトークン予測」のみです。チャットボットにするにはアライメントが必要。

\keyterm{SFT (Supervised Fine-Tuning)}：コマンドと応答のペアで指導学習。基本チャットボット行動学習。

\keyterm{RLHF}: 報酬モデル+ PPOで好みを最適化。 ChatGPTが使用。

\keyterm{DPO (Direct Preference Optimization)}：補償モデルなしで直接の好みを最適化します。シンプルで効果的。 Zephyr、Llama-2-Chatを使用。

\keyterm{LoRA}：低ランクアダプタでファインチューニングコスト99\%節約。 7Bモデルを200MBアダプターでファインチューニング。

\section{データパイプライン}

\keyterm{品質フィルタ}：長さ、言語、繰り返し率で低品質のテキストを削除します。

\keyterm{MinHash}：Jaccard類似度近似による重複文書の検出。 Web データの 30$\sim$50\% が重複しています。

\keyterm{合成データ}：強力なモデルで特定のドメインデータを生成します。 Evol-Instruct、Self-Instructなど。

\section{2巻学習準備}

2巻は1巻のコードに基づいて拡張する。 1冊のGPTモデルをインポートします。
\begin{itemize}
\item 分散学習で包み（2$\sim$第4章）
\item アテンションをGQAに置き換えて（5章）
\item FFNをSwiGLUに置き換え（5枚）
\item LoRAアダプターを追加します（第8章）。
\end{itemize}

1冊を慣れて扱えば2冊も自然に追いつくことができる。 1冊のコードを直接修正してみて実験することをお勧めします。

\section{推奨学習パス}

\begin{enumerate}
\item 1巻のコードで小さなモデルを学ぶ（\code{scripts/tiny\_train.py}）。
\item 自分の言葉で実験（ウィキバックと韓国語、GitHubコードなど）。
\item 2巻に渡って分散学習環境を構築。
\item LoRAによるファインチューニング実験（\code{scripts/lora\_finetune.py}）。
\item 量子化による推論の最適化（\code{scripts/quantize\_demo.py}）。
\item 実際のHuggingFaceモデルでサービング練習（vLLM、TGI）。
\end{enumerate}

このパスをたどると、ChatGPTレベルのLLMを理解し運営する能力があります。コーディングを停止しないでください。実験を止めないでください。それが実力を育てる唯一の道だ。
